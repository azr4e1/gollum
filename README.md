# goLLuM

Small library for LLM completion. It provides a single client to interface to multiple LLM providers.

## LLM providers supported:

- OpenAI:
  - [X] Completion
  - [X] TTS
  - [ ] Embeddings
- Ollama:
  - [ ] Completion
  - [ ] Embeddings
- Gemini:
  - [ ] Completion
  - [ ] Embeddings
- Claude
  - [ ] Completion
  - [ ] Embeddings
